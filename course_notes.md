## TDDE52 Programming Project with Open Source Code
- Examiner: Daniel Varro
- Faculty: Faculty of Science and Engineering
- Period/Module: Periods 1-2, Timetable module 4 (Autumn 2026)
- Summary: Team-based contribution to an external open-source project across the whole semester, choosing either a large-scale engineering focus or an algorithmic contribution focus while practicing industrial tools and coordination.

- Examination: PRA1 Project 6 ECTS (U,3,4,5) assessed via oral individual review of Git-based contributions, documentation, and presentations; no separate written exam.
- Credits split: Written 0 / Lab 0 / Project 6 / Oral 0 (oral defense embedded).
